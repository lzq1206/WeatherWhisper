# 云量五档分布升级审计（2026-07-17）

## 任务目标

将原“逐日平均云量”单曲线升级为接近 WeatherSpark 表达方式的五档连续堆叠分布：晴天、大部分晴天、部分多云、大部分多云、阴天。

## 输入与口径

- 城市范围：线上全部 339 个省级、地级与特别行政区条目。
- 数据输入：本地冻结的 NASA POWER/MERRA-2 1991–2020 逐日 `CLOUD_AMT` 缓存。
- 分类阈值：`[0,20)` 晴天、`[20,40)` 大部分晴天、`[40,60)` 部分多云、`[60,80)` 大部分多云、`[80,100]` 阴天。
- 平滑方法：先按每个历史日的日均天空覆盖率分类，再按年内日期汇总，并采用 15 日居中环形窗口平滑。
- 黑色分界线：晴天、大部分晴天与部分多云合计，即日均云量低于 60% 的天气日占比。
- 与 WeatherSpark 的差异：本站为逐日平均云量类别的天气日占比；WeatherSpark 参考图为逐小时云量类别占比，二者不应解释为完全相同的统计量。

## 执行方法

```bash
python3 scripts/add_cloud_category_climatology.py \
  --root . \
  --cache-dir /tmp/weatherwhisper_climate_normals_cache
npm run build
```
