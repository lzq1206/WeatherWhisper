# Prefecture-level admin unit expansion validation (2026-07-11)

## Scope

WeatherWhisper was expanded from the prior 293-prefecture-city catalog to the full mainland China prefecture-level administrative-unit catalog:

- 293 地级市
- 7 地区
- 30 自治州
- 3 盟
- Total: 333 units

Province-level municipalities, county-level cities, urban districts, counties, flags and special administrative regions are excluded from the 333-unit map catalog.

## Data and throttling method

- Admin-unit list source: `https://zh.wikipedia.org/wiki/中华人民共和国地级行政区列表`
- Weather station source: `https://climate.onebuilding.org/WMO_Region_2_Asia/CHN_China/`
- Metadata enrichment uses batched MediaWiki API calls, up to 40 titles per request, with retry/backoff and a pause between batches.
- Climate ZIP downloads are deduplicated by station file and skipped when corresponding EPW files already exist locally.
